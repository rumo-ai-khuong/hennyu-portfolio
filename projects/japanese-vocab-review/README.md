# Japanese Vocab Review

日本語の語彙をPythonで整理するための小さな練習プロジェクトです。

## 目的

- Pythonの基本文法を練習する
- 日本語の語彙を整理する
- 授業で学んだ言葉を復習する
- AI・IT学習ポートフォリオの最初の小さなプロジェクトにする

## 最初に使う語彙

| 日本語 | 読み方 | 意味 |
|---|---|---|
| 経営理念 | けいえいりねん | triết lý kinh doanh |
| 経営戦略 | けいえいせんりゃく | chiến lược kinh doanh |
| コンプライアンス | こんぷらいあんす | tuân thủ pháp luật |
| CSR | しーえすあーる | trách nhiệm xã hội của doanh nghiệp |
| BCP | びーしーぴー | kế hoạch duy trì hoạt động kinh doanh |
| BCM | びーしーえむ | quản lý duy trì hoạt động kinh doanh |

## 次にやること

- `main.py` を作成する
- 語彙リストをPythonで表示する
- 少しずつ語彙を追加する
---

## ファイル構成

```text
japanese-vocab-review/

```

## 使い方

ターミナルで以下のコマンドを実行します。

```bash
python main.py
```
